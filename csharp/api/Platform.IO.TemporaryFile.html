    Type Name Description
    Boolean manual

    A Boolean value that determines whether the disposal was triggered manually (by the developer's code) or was executed automatically without an explicit indication from a developer.

    Значение типа Boolean, определяющие было ли высвобождение вызвано вручную (кодом разработчика) или же выполнилось автоматически без явного указания со стороны разработчика.
    Boolean wasDisposed

    A Boolean value that determines whether the TemporaryFile was released before a call to this method.

    Значение типа Boolean, определяющие были ли освобождены ресурсы, используемые TemporaryFile до вызова данного метода.
