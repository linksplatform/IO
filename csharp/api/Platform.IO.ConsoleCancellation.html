    Class ConsoleCancellation

    Represents the class that simplifies the console applications implementation that can be terminated manually during execution.

    Представляет класс, упрощающий реализацию консольных приложений, выполнение которых может быть прекращено в процессе выполнения вручную.

    Dispose(Boolean, Boolean)

    Unsubscribes from the CancelKeyPress event and attempts to dispose the CancellationTokenSource.

    Отписывается от события CancelKeyPress и пытается высвободить ресурсы, используемые CancellationTokenSource.

    Parameters
    Type Name Description
    Boolean manual

    A Boolean value that determines whether the disposal was triggered manually (by the developer's code) or was executed automatically without an explicit indication from a developer.

    Значение типа Boolean, определяющие было ли высвобождение вызвано вручную (кодом разработчика) или же выполнилось автоматически без явного указания со стороны разработчика.
    Boolean wasDisposed

    A Boolean value that determines whether the ConsoleCancellation was released before a call to this method.

    Значение типа Boolean, определяющие были ли освобождены ресурсы, используемые ConsoleCancellation до вызова данного метода.
