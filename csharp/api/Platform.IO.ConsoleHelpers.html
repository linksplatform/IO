    Debug(String, Object[])

    Writes text representations of the specified args array objects to the standard output stream using the specified format, followed by the current line terminator.

    Записывает текстовые представления объектов заданного массива args, в стандартный выходной поток с использованием заданного format, за которым следует текущий признак конца строки.

    Declaration
    [Conditional("DEBUG")]
public static void Debug(string format, params object[] args)
    Parameters
    Type Name Description
    String format

    The composite format string.

    Строка составного формата.
    Object[] args

    The object array to write to the standard output stream using format.

    Массив объектов для записи в стандартный выходной поток с использованием format.
    Remarks

    The method is only executed if the application was compiled with the DEBUG directive.

    Метод выполняется только если приложение было скомпилировано с директивой DEBUG.

    GetOrReadArgument(Int32, String, String[])

    Gets an argument's value with the specified index from the args array and if it's absent requests a user to input it in the console.

    Получает значение аргумента с указанным index из массива args, a если оно отсутствует запрашивает его ввод в консоли у пользователя.

    Declaration
    public static string GetOrReadArgument(int index, string readMessage, params string[] args)
    Parameters
    Type Name Description
    Int32 index

    The ordinal number of the argument in the array.

    Порядковый номер аргумента в массиве.
    String readMessage

    The message's text to a user describing which argument is being entered at the moment. If the args array doesn't contain the element with the specified index, then this message is used.

    Текст сообщения пользователю описывающее какой аргумент вводится в данный момент. Это сообщение используется только если массив args не содержит аргумента с указанным index.
    String[] args

    The argument array passed to the application.

    Массив аргументов переданных приложению.
    Returns
    Type Description
    String

    The value with the specified index extracted from the args array or entered by a user in the console.

    Значение с указанным index, извлечённое из массива args, или введённое пользователем в консоли.
